# 🚀 מוריד המדיה האוניברסלי

אפליקציה להורדת וידאו ואודיו מפלטפורמות שונות באיכות גבוהה.

## ✨ תכונות

- 🎬 הורדת וידאו באיכויות שונות (360p עד 4K)
- 🎵 הורדת אודיו בפורמטים MP3, M4A, WAV
- 📝 תמיכה בהורדת כתוביות
- 🌍 תמיכה במאות אתרים: YouTube, TikTok, Instagram, Twitter/X, Facebook ועוד
- 🌙 עיצוב מודרני ונוח לשימוש
- 🇮🇱 ממשק בעברית

## 🔗 פלטפורמות נתמכות

YouTube • TikTok • Instagram • Twitter/X • Facebook • Vimeo • Twitch • Reddit • Dailymotion ומאות אתרים נוספים

## 🚀 שימוש

1. הדבק קישור לסרטון
2. לחץ "בדוק קישור"
3. בחר איכות (וידאו או אודיו)
4. לחץ "הורד"
5. לחץ על כפתור ההורדה לשמירה במחשב

## 📦 התקנה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ פריסה ב-Streamlit Cloud

1. העלה את הקבצים ל-GitHub
2. היכנס ל-[share.streamlit.io](https://share.streamlit.io)
3. חבר את ה-repository
4. לחץ Deploy!
